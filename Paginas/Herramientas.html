Herramientas utilizadas

Visual studio Code

Visual Studio Code (VS Code) es un editor de código fuente gratuito desarrollado por Microsoft para Windows, Linux y macOS. Proporciona un entorno ligero pero potente para programar, depurar y controlar versiones. Lo que nos ayuda es poder nuestras pestañas con archivos de tipo js, php entre hojas de estilo, entre otras.

Xampp

XAMPP es un paquete de software gratuito y de código abierto que facilita la creación de un entorno de desarrollo local para sitios web. El nombre "XAMPP" es un acrónimo que representa los componentes principales del paquete:

  • X: Refiere a cualquier sistema operativo (Windows, Linux, macOS).
  • A: Apache, que es un servidor web.
  • M: MySQL, que es un sistema de gestión de bases de datos.
  • P: PHP, Perl o Python, que son lenguajes de programación utilizados para el desarrollo web.

Git Hub

La importancia de Git Hub fue al momento de agregar el repositorio ademas de subir la aplicacion para que pueda ser accesible a cualquier persona que contenga wel link. Es una plataforma de desarrollo colaborativo basada en Git. Git es un sistema de control de versiones que permite a los desarrolladores gestionar y realizar un seguimiento de los cambios en el código fuente durante el desarrollo de software. GitHub proporciona una interfaz web y herramientas adicionales que facilitan la colaboración y el trabajo en equipo.

  • Control de Versiones Distribuido: GitHub utiliza Git para el control de versiones, permitiendo a los desarrolladores trabajar de manera colaborativa y mantener un historial completo de cambios en el código fuente.
  • Repositorios: Los proyectos en GitHub se almacenan en lo que se llama repositorios. Un repositorio es un lugar donde se almacena y gestiona el código fuente, y puede incluir archivos, carpetas, imágenes y documentación.
  • Colaboración: GitHub facilita la colaboración en equipo. Varios desarrolladores pueden trabajar en un proyecto al mismo tiempo, y Git se encarga de fusionar los cambios de manera eficiente.

Git Bash

Es una aplicación que proporciona una interfaz de línea de comandos (CLI) para interactuar con Git en sistemas operativos Windows. Aunque Git Bash incluye el término "Bash" en su nombre, no es simplemente una interfaz para el shell Bash, sino que proporciona un entorno similar al de la línea de comandos de Unix/Linux en sistemas Windows.

Se utilizo para poder subir al repositoro el contenido del sitio web con comandos que se muestran a continuación.

  • Interfaz de Comandos Git: Git Bash proporciona una interfaz de línea de comandos para ejecutar comandos Git en sistemas Windows.
  • Shell Unix-like: Incluye un entorno similar a un shell Unix-like, permitiendo el uso de comandos de estilo Unix/Linux en sistemas Windows.
  • Scripts y Automatización: Puedes escribir scripts de shell en Git Bash para automatizar tareas relacionadas con Git.
  • Acceso a Herramientas de Git: Proporciona acceso a todas las herramientas de Git, incluso aquellas que no están disponibles en la interfaz gráfica de usuario.
  • Compatibilidad con SSH: Git Bash es compatible con SSH, permitiendo la autenticación y conexión a repositorios remotos de forma segura.
