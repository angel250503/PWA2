Páginas Web Progresivas

Es importante mencionar desde donde surge la tecnología y esto comienza allá por la Navidad de 1990, Sir Tim Berners-Lee y su equipo del CERN construyeron todas las herramientas necesarias para una web que funcione. Crearon HTTP, HTML y el mundo primer navegador web, llamado WorldWideWeb

Las PWA cambian las reglas del juego. Los principales proveedores de navegadores han estado trabajando juntos para mejorar la forma en que construimos para la web y hemos creado un nuevo conjunto de características que brindan a los desarrolladores web la capacidad de crear sitios web rápidos, confiables y atractivos. Una PWA debería ser todas estas cosas:

  • Responsivo

  • Independiente de la conectividad

  • Interactivo con la sensación de una aplicación nativa

  • Siempre actualizado

  • Seguro

  • Confiable

Significa que podemos empezar a crear sitios web que puedan hacer frente a los cambios. Condiciones de red o conectividad nula. También significa que podemos crear sitios web que son más atractivos para nuestros usuarios y les ofrecen una experiencia de navegación de primera clase.

Ventajas y Desventajas de las PWA

Ventajas Desventajas
Acceso sin conexión: Las PWAs pueden funcionar sin conexión a Internet o en conexiones de red débiles, ya que pueden almacenar datos en caché. Limitaciones de API: Algunas APIs nativas pueden no estar disponibles para PWAs, lo que podría limitar ciertas funcionalidades avanzadas.
Instalación sin tienda: Los usuarios pueden instalar una PWA directamente desde el navegador, sin necesidad de pasar por una tienda de aplicaciones. Menor integración con el sistema: Las PWAs pueden tener restricciones en cuanto a la integración profunda con el sistema operativo y el hardware del dispositivo en comparación con las aplicaciones nativas.
Actualizaciones automáticas: Las PWAs se actualizan automáticamente, lo que garantiza que los usuarios siempre tengan la última versión sin tener que descargar manualmente actualizaciones. Consumo de recursos: En comparación con las aplicaciones nativas, las PWAs pueden consumir más recursos en términos de memoria y rendimiento.
